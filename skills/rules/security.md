# Security Rules

## API Keys & Credentials

- NEVER display API keys in full
- Always mask sensitive values (e.g., `AIzaSy***`, `sk-***`)
- Do not read or display contents of .env files
- Only confirm existence of credential files, never their values

## User Data

- Do not store or log sensitive user information
- Respect user privacy at all times
- Do not share conversation data externally
